# Babcock University Curriculum & Course Database (Computer Science / Software Engineering)

## Core Level 200L Courses
* **COSC 200 - Web Application Development**
  * **Units:** 3
  * **Lecturer:** Adams P.L.
  * **Prerequisites:** None
  * **Description:** Designing and implementing interactive, secure modern web applications.

* **COSC 202 - Advanced Database Management Systems**
  * **Units:** 3
  * **Lecturer:** Johnson R.T.
  * **Prerequisites:** Introduction to Database Systems
  * **Description:** SQL optimization, transaction management, NoSQL systems, database scaling.

* **COSC 204 - Designing and Development of Object Oriented Software**
  * **Units:** 2
  * **Lecturer:** Maitanmi S.O.
  * **Prerequisites:** Introduction to Programming
  * **Description:** OO principles, inheritance, polymorphism, design patterns.

* **COSC 268 - Mobile Application Development**
  * **Units:** 3
  * **Lecturer:** Peterson J.E.
  * **Prerequisites:** COSC 204
  * **Description:** Building native and cross-platform apps for iOS and Android.

* **COSC 872 - Data Structures and Algorithms**
  * **Units:** 2
  * **Lecturer:** Singh R.V.
  * **Prerequisites:** None
  * **Description:** Linked lists, trees, graphs, sorting, searching, time complexity analysis.

* **SENG 270 - Computer Networks**
  * **Units:** 3
  * **Lecturer:** Rodriguez T.F.
  * **Prerequisites:** None
  * **Description:** TCP/IP model, routing protocols, network security, packet switching.

* **ITGY 271 - Human-Computer Interaction**
  * **Units:** 3
  * **Lecturer:** Lee M.N.
  * **Prerequisites:** None
  * **Description:** UI/UX design principles, usability testing, prototyping, user-centered design.

* **BU-GST 201 - Artificial Intelligence Principles**
  * **Units:** 4
  * **Lecturer:** Nguyen A.K.
  * **Prerequisites:** None
  * **Description:** Machine learning, neural networks, search algorithms, logic reasoning.

* **BU-GST 205 - Software Engineering**
  * **Units:** 4
  * **Lecturer:** Williams H.J.
  * **Prerequisites:** None
  * **Description:** Software development life cycles, agile methodologies, requirements gathering, system architecture.
